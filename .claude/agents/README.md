# Tým agentů — stunova

| Agent | Zaměření v projektu |
|---|---|
| produktovy-manazer | Scope 3 povrchů (web/portál/admin), priorizace |
| business-analytik | Specifikace klientského portálu a admin workflow |
| softwarovy-architekt | Architektura, Clerk auth, datový model |
| backend-vyvojar | API, DB, integrace, auth (Clerk) |
| frontend-vyvojar | Next.js stránky, MDX, komponenty |
| devops-inzenyr | Vercel deploy, CI/CD, prostředí |
| security-specialista | Ochrana klientských/finančních dat, auth review |
| qa-tester | Test plány portálu i adminu, regrese |
| ux-designer | Flows klientského portálu, IA, wireframy |
| ui-designer | Vizuální design obrazovek, komponenty, stavy |
| jack-dorsey-ux | Polishing, minimalismus, browser ověření |
| brand-designer | Osobní brand účetní — identita, logo, paleta |
| web-designer | Konverzní marketing landing, CTA |
| marketer | Akvizice klientů, positioning, kanály |
| copywriter | Texty webu, microcopy portálu, e-maily |
